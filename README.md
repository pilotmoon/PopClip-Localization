# PopClip Translations

This repository copntains [PopClip](https://www.popclip.app)'s user interface strings and their translations in various languages.

The original development language for PopClip is English (`en`).

The translations have been created piecemeal over the course of many years by various processes: commerical translation servies, volunteer translators, and machine translation. This was previously hosted on Crowdin but I have moved them here for a simpler approach (June 2026).
